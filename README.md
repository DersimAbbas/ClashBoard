# ClashBoard

Fighting game league standings and leaderboard application.

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Backend:** .NET 9 Web API + Entity Framework Core
- **Database:** PostgreSQL 17
- **Infrastructure:** Docker Compose

## Getting Started

```bash
# Start PostgreSQL
docker compose up db -d

# Run the API
cd src/ClashBoard.Api
dotnet run

# Run the frontend
cd client
npm run dev
```
